Aliran pemrosesan berkesinambungan dari tangkapan foton kamera hingga eksekusi perintah OS dalam latensi < 16.5 ms.

📷
1. Akuisisi & Preprocessing
Webcam 720p menangkap frame RGB. Dilakukan Mirror Flip horizontal agar interaksi terasa alami layaknya cermin.
➡️
🖐️
2. Spasial Ekstraksi (3D)
MediaPipe mengekstrak koordinat $(X, Y, Z)$ dari 21 titik persendian per tangan. Koreksi label lateralitas fisik.
➡️
📐
3. Vektorisasi & Scaler
Penggabungan koordinat tangan Kiri ($63$) + Kanan ($63$) = vektor $126$ dimensi. Dinormalisasi via `StandardScaler`.
➡️
⚡
4. Prediksi SVM Brain
Model SVM RBF mengevaluasi vektor spasial dan mengembalikan ID gestur ($0 - 7$) dengan probabilitas keyakinan tinggi.
➡️
🖥️
5. Action Dispatcher
Eksekusi aksi OS: EMA Move, Click/Double Click, Scroll Vert/Horiz, Screenshot, atau pemanggilan Hologram Keyboard.

Konstruksi Matriks Fitur 126-Dimensi

Sistem menyusun vektor tunggal $X_{input}$ dengan menggabungkan koordinat tangan Kiri ($L$) dan Kanan ($R$): 

X_input = [
  L_x0, L_y0, L_z0, ..., L_x20, L_y20, L_z20,  # 63 fitur Kiri
  R_x0, R_y0, R_z0, ..., R_x20, R_y20, R_z20   # 63 fitur Kanan
] # Total: 126 elemen floating point

Zero-Padding Strategy: Jika pengguna hanya mengangkat satu tangan (misal hanya tangan kanan aktif), maka 63 fitur tangan kiri secara otomatis diisi nilai `0.0` agar dimensi input model tetap stabil pada $126$ fitur tanpa gangguan dimensi.

Otak AI: Support Vector Machine (SVM RBF)

Mengapa vektor 126-dimensi + SVM jauh lebih superior dibanding CNN gambar 2D untuk kontrol kursor real-time?

Konfigurasi Hiperparameter Model (`SVC`)
  • `kernel='rbf'` (Radial Basis Function): Mampu memisahkan batas keputusan gestur tangan yang bersifat non-linear kompleks dalam ruang multi-dimensi melalui pemetaan kernel hiper-bola berpresisi tinggi.
  • `C=10`: Parameter penalti regularization yang kuat untuk meminimalkan misklasifikasi pada ruang vektor spasial dan menjaga ketajaman batas antar-gestur.
  • `class_weight='balanced'`: Bobot otomatis penyeimbang kelas untuk memastikan sensitivitas AI tetap sempurna baik pada gestur dominan maupun gestur khusus berfrekuensi cepat.
  • `gamma='scale'`: Penyesuaian varians data spasial yang secara otomatis menyesuaikan rentang koordinat 3D yang telah dinormalisasi `StandardScaler`.

Spesifikasi Klasifikasi Gestur (Label 0 – 3)

Pemetaan logika aksi sistem operasi berbasis aturan untuk empat kelas gestur utama pertama (Diperbarui).

Label 0 • 19.4% Data
Telunjuk Lurus (Move)
Aksi OS: `pyautogui.moveTo(x, y)`
Geometri: Ujung telunjuk kanan (Landmark 8) lurus, jari lain menekuk.
Filter: Exponential Moving Average (`alpha=0.25`) untuk menghapus getaran natural tangan.
Label 1 • 20.3% Data
Cubit / Pinch (Clicking & Drag)
Aksi OS: Click, Double Click, & Hold-Drag
Geometri: Jarak 3D Jempol-Telunjuk < `0.055`.
Logika Temporal: Dilepas < 0.28s = Klik Kiri; Klik 2x cepat (< 0.65s) = Double Click; Ditahan > 0.28s = Drag Kursor; Cubit Jari Tengah = Klik Kanan.
Label 2 • 14.0% Data
C-Shape (Volume & Brightness)
Aksi OS: Volume Up/Down & WMI Brightness
Geometri: Rotasi pergelangan `atan2(delta_y, delta_x)` dengan threshold delta > `0.08` rad.
Bimanual: Tangan Kanan = Volume, Tangan Kiri = Brightness (Asinkron).
Label 3 • 3.3% Data
Dual V-Sign (Hologram KB)
Aksi OS: Toggle Virtual Keyboard
Geometri: Kedua tangan mengacungkan V-Sign (Peace) secara simultan.
Logika Temporal: Ditahan selama 2.0 detik untuk memunculkan antarmuka pengetikan virtual.

Spesifikasi Klasifikasi Gestur (Label 4 – 7)

Pemetaan logika navigasi tingkat lanjut, screenshot bimanual, lateral scroll, dan pengamanan sistem (Diperbarui).

Label 4 • 3.5% Data
Frame Kamera (Screenshot)
Aksi OS: `pyautogui.screenshot(region)`
Geometri: Bounding box dari ujung telunjuk & jempol kedua tangan membentuk bingkai.
Logika Temporal: Hitung mundur 1.5 detik dengan visual progress bar + animasi shutter flash.
Label 5 • 22.2% Data
Ninja Dua Jari (Scroll Vert)
Aksi OS: Vertical `scroll`
Geometri: Telunjuk & tengah sejajar.
Logika: Pergeseran vertikal `dy` atau horizontal `dx` > `0.015` dikalikan faktor akselerasi `2500` untuk guliran dokumen mulus dan responsif.
Label 6 • 8.2% Data
Tangan Kertas (Horizontal Swipe)
Aksi OS: Horizontal Scroll (`hscroll`)
Geometri: Telapak tangan terbuka (Open Palm).
Logika Baru: Pergeseran horizontal `dx` pada landmark MCP `[9]` dikalikan faktor `3000` (`pyautogui.hscroll(dx*3000)`), memungkinkan geser ke kanan/kiri pada spreadsheet/timeline super mulus!
Label 7 • 9.2% Data
Mengepal / Fist (Standby Lock)
Aksi OS: System Lock / Unlock
Geometri: Seluruh jari menggenggam erat (Fist ✊).
Logika Temporal: Ditahan 1.0 detik untuk membekukan semua input mouse dengan overlay merah peringatan non-blocking.
